class jala.ListRenderer


This class eases the rendering of arbitrary lists. Such lists can be represented by HopObject collections or Arrays of HopObjects. Various arguments provide customization of the number of items per page, the maximum of pages and so on.

Constructor Detail

jala.ListRenderer

jala.ListRenderer(<Object> listParam, <Object> renderers)
    Construct a list renderer.
    Parameters:
      listParam - Customization parameters:
      • collection - Collection or Array of HopObjects.
      • currentPage - Optional index of the current page (req.data.page)
      • itemsPerPage - The optional number of items per list page.
      • maxPages - The optional maximum number of pages.
      • href - An optional URL string to link to.
      • urlParams - Optional extra URL parameters.
      renderers - The list types and renderer methods used to render the single items in a list.
    Returns:
      A new list renderer instance.
    Throws:
      - Exception if an ArrayList containing a subset of a bigger Array and has too view elements for one page or the last page.
