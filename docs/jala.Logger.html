void logException(<Object> ex, funcName, <String> userName, <String> reqPath)
    Writes an exception to application log. This method accepts both Java exceptions and Javascript objects thrown as Exception.
    Parameters:
      ex - The exception object
      userName - The name of the user that issued the request (optional)
